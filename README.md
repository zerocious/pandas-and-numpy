# Pandas and NumPy

Practice notebook and datasets for NumPy and pandas exercises.

## Project layout

```
pandas-and-numpy/
├── notebooks/
│   └── NumPy.ipynb          # Main exercises (NumPy + pandas)
├── data/
│   ├── raw/                 # Input CSV files for exercises
│   └── output/              # Results saved from the notebook
├── notes/                   # Study notes and reflections
├── README.md
└── .gitignore
```

## Usage

1. Open `notebooks/NumPy.ipynb` in Jupyter or VS Code.
2. Run cells in order. Paths assume the notebook lives in `notebooks/`.
3. New exports are written to `data/output/`.

## Requirements

- Python 3.10+
- `numpy`
- `pandas`

```bash
pip install numpy pandas
```
